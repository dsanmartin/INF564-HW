Tarea 2 - Daniel San Martín

1. COMPILACION:
------------------
Realizar make adentro de la carpeta.

2. EJECUCION:
------------------
2.1 Problema 1

./tarea2 alg < input_file
* alg: 0 Para el algoritmo fuerza bruta y 1 para el algoritmo propuesto.
* input_file: ruta del archivo de entrada.
Formato input_file segun lo detallado en el enunciado de la tarea.


2.2 Problema 2

./insertion OPT N SHOW < input_file
* OPT: Version del algoritmo. 0 Original, n para version n, 5 Library Sort
* N: Tamaño del arreglo a ordenar.
* SHOW: Show sorted array.
* input_file: ruta del archivo de entrada.
Formato input_file:
entero_1
entero_2
...
entero_N
